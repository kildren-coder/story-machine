## 问的是什么

脱离 Gemini 免费档后换什么提取引擎?案头查五项:开源 API 价格(海外/国内)、国内托管审查、opencode Go、Gemini 数据条款、本地约束解码(实测留 #14)。

## 答案是什么

成本非问题、合规是。海外开源按量付费(DeepInfra/Together)全管线仅 $2–6/月;国内托管一票否决(法律强制过滤、~85% 拒答);opencode Go 因 ToS 禁「程序化提取+未登录进程」不建议;Gemini 免费档拿数据训练+人审。约束解码 schema 已成熟、幻觉率未解。
